# customize

> [!info] Surgical section updates — requires a NAMED target
> Bare verbs without targets ("update everything", "fix it") route to [[advise]] for clarification, never here. Customize requires you to say WHAT to change.

## When this fires

Triggers: "update the products section", "tweak the directives", "change the routing table", "modify the entity facts" — always with a named section.

## What it does

1. Loads the existing file
2. Targets the named section
3. Applies the requested edit
4. Re-validates spec compliance + size
