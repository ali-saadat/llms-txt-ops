# sector router

> [!note] Composition: product-domain × delivery-channel
> A single site can be EITHER a product domain (legal, healthcare, real-estate) AND a delivery channel (marketplace, b2b-saas). The router composes both.

## Common combinations

| Combination | Approach |
|---|---|
| Legal-tech B2B SaaS | [[b2b-saas]] skeleton + [[legal-services]] directives |
| Health-tech B2B SaaS | [[b2b-saas]] skeleton + [[healthcare]] safety directives |
| Fintech B2B SaaS | [[b2b-saas]] skeleton + [[fintech]] disclaimers |
| EdTech B2B SaaS | [[b2b-saas]] skeleton + [[education]] verify-current directives |
| PropTech marketplace | [[marketplace]] skeleton + [[real-estate]] Fair Housing |
| Auto marketplace | [[marketplace]] skeleton + [[automotive]] recall disclaimers |
| Hospitality marketplace | [[marketplace]] skeleton + [[hospitality]] rate-volatility |
| News + media aggregator | [[news-publisher]] editorial + [[media-entertainment]] licensing |
